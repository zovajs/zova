# Cabloy Basic

This suite `zova-suite-cabloy-basic` uses the `Daisyui + Tailwindcss` UI library and provides a set of out-of-the-box frontend modules to accelerate the development of fullstack projects such as `Admin-Dashboards`

## Module List

### 1. Core Modules

| Name           | Description                                                                                                                                                 |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| basic-adapter  | UI component adapter: defines commonly used UI components for rendering `Table/Form`                                                                        |
| basic-restpage | Dynamically render CRUD list pages, entry pages, and search forms, while demonstrating best practices for Tanstack Table, Tanstack Form, and Tanstack Query |

### 2. Other Modules

| Name        | Description                                                                                 |
| ----------- | ------------------------------------------------------------------------------------------- |
| basic-date  | Date range component, automatically handles `Timezone`                                      |
| basic-form  | Form behavior component, controls form rendering behavior based on the `Behavior` mechanism |
| basic-table | Table component implemented based on `Tanstack Table`                                       |
